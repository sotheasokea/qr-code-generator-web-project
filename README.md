# QR Code Generator Web App

A simple, client-side web application for generating QR codes from user-provided URLs.
---
**Live Demo**: https://qr-code-generator-eight-neon.vercel.app/

## Features
- **Dynamic Generation:** Generates QR codes instantly in the browser using JavaScript.
- **URL Input:** Allows users to input a URL and generate the corresponding QR code.
- **Download Functionality:** Provides an easy way for the user to download the generated QR code as a PNG image.
- **Client-Side Only:** The entire functionality runs in the user's browser, requiring no server-side setup.

## Installation
This project is a client-side script and requires no installation steps beyond opening the HTML file in a web browser.

1.  Ensure you have an HTML file (e.g., `index.html`) that links to the necessary JavaScript and includes the necessary DOM elements for this script to function correctly.
2.  Open the HTML file in any modern web browser.

## Usage
1.  Navigate to the generated page.
2.  Enter the desired URL into the input field.
3.  Click the "Generate QR Code" button to create the code dynamically.
4.  Click the "Download QR Code" button to save the image to your device.

## Technology
*   **HTML/CSS:** For the user interface structure and styling.
*   **JavaScript:** Handles all dynamic logic, including library loading (`qrcodejs`), QR code generation, and file download handling.
*   **QRCode.js:** The external JavaScript library used for the actual QR code encoding.
